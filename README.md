Team Er404 Submission for Dell Hackathon

# Deployment (Chrome)
The extension is not yet completed / published. To test it you can download the extension folder [here](https://github.com/PolarisedBear/P.A.T.C.H-Extension-and-Support-App/releases/tag/prototype_v1.1), and then in *chrome://extensions*, click <b>Load Unpacked</b> and select the extension/dist folder.

## UPDATE 12/3/26:
Shifted to a bundled webp backend. In terminal, type\
```$env:NODE_ENV="development"```\
to allow debugging, then enter\
```npm run build ```\
to build.
